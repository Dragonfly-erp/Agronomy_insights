# Візуальна галерея кейсу

Це текстові візуали, які можна відкрити на GitHub і заскрінити для відео. Сирі скріншоти поля не додані, щоб не показувати локацію та дані господарства.

## 1. Фінальна нормалізована карта, спрощений вигляд

Легенда: 🟩 висока врожайність, 🟨 середня, 🟧 нижча, 🟥 проблемні або нижчі ділянки.

```text
                🟨🟨🟩🟩🟨🟧
            🟨🟩🟩🟩🟨🟨🟧🟧
        🟨🟩🟩🟨🟨🟩🟩🟨🟧
      🟨🟨🟩🟩🟩🟨🟨🟨🟧🟥
    🟧🟨🟨🟩🟩🟩🟨🟨🟧🟧
   🟥🟧🟨🟨🟩🟩🟨🟨🟨🟧
    🟧🟨🟩🟩🟩🟨🟨🟧🟧
      🟨🟨🟩🟩🟨🟧🟧🟥
        🟧🟨🟨🟩🟨🟧🟥
```

Сенс: фінальна карта показує неоднорідність поля після виправлення геометрії, очищення аномалій, інтерполяції провалів і калібрування до контрольної середньої.
