# Тарас

Кейс по карті врожайності SMS / Ag Leader.

У папці мають зберігатися тільки публічно безпечні матеріали: фінальна нормалізована карта, PDF/PNG для пояснення, ZIP із робочим shapefile і короткі нотатки до відео.

Сирі дані господарства, координати, назви поля та скріншоти з видимою локацією краще не публікувати без окремого дозволу.
