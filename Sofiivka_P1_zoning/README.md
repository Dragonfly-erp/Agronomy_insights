# Зонування поля П1 (Софіївка) — Геворг ПОСП

Супутникове зонування поля (~86 га) за NDVI для **змінної норми внесення добрив**
та **прив'язаного до зон відбору ґрунтових зразків**.

## 📄 Почніть звідси
**[report/ЗВІТ_зонування_П1_Софіївка.pdf](report/ЗВІТ_зонування_П1_Софіївка.pdf)**
(або [той самий звіт у Markdown](report/ЗВІТ_зонування_П1_Софіївка.md))

## Головне
- **Робочі зони — за 2025 роком** (повний покрив): `shapefiles/zones/zones_2025.shp` — **5 зон**
  (Зона 1 = найслабша … Зона 5 = найсильніша). Варіанти: `_k4`, `_k6`.
- **2024 рік — голий ґрунт**, не корелює з урожайністю (r=0.04) → у продуктивні зони
  не включено; збережено окремо як `shapefiles/zones/soil_2024.shp`.
- **Перевірка рельєфом:** продуктивність ↔ висота = −0.72 (нижче = продуктивніше);
  зони відокремлені на 93.3% (VR-індекс).
- **Рельєф у shape:** горизонталі 1 м, класи висот, крутість, топопозиція — `shapefiles/relief/`.
- **Відбір зразків:** 15 точок (композит по 15–20 уколів, 0–30 см) — `shapefiles/sampling/`.

## Формати
Кожен шейпфайл — у двох CRS: **UTM 36N (EPSG:32636)** метри (без суфікса) та
**WGS84 (EPSG:4326)** (`_wgs84`). Растри — GeoTIFF (UTM 36N).

Джерело рельєфу: **Copernicus GLO-30 DEM** (ESA, безкоштовно). Конвеєр обробки —
`scripts/` (Python, відтворюваний).
